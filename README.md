# 🌊 WalletWave — Personal Finance Tracker

WalletWave is a modern and responsive Personal Finance Tracker built using HTML, CSS, and JavaScript.

It helps users manage their daily expenses, track budgets, visualize spending analytics, and export expense reports easily.

---

# 🚀 Features

✅ Add Expenses  
✅ Edit Expenses  
✅ Delete Expenses  
✅ Monthly Budget System  
✅ Expense Analytics Chart  
✅ Search Expenses  
✅ Filter by Category  
✅ Export Expenses as CSV  
✅ Dark / Light Mode  
✅ Responsive Design  
✅ Local Storage Support  
✅ Toast Notifications  
✅ Smooth Animations  
✅ Interactive Sidebar Navigation
